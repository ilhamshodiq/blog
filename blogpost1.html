Instalasi OS Proprietary

Langkah-langkah penginstalan windows 10

Windows adalah sebuah Operation sistem (OS) ciptaan perusahaan ternama microsoft. Windows sekarang ini adalah operasi sistem yang berbasis GUI tentunya. Sebelumnya windows berevolusi dari OS yang menggunakan command-line dan text saja, yaitu MS-DOS. Windows GUI pertama dirilis pada tahun 1983 dan perkembangannya pesat sampai versi windows sekarang yaitu windows 11.

Pada blog ini kita akan membahas cara penginstalan salah satu versi dari windows, yaitu windows 10.

Windows 10 sendiri telah dirilis sejak tahun 2015, dan saat ini masih menjadi OS yang paling banyak digunakan dan diminati di dunia. Mengesampikan update windows versi terbaru 5 oktober kemarin yaitu windows 11.



Siapkan media bootable untuk install

Untuk mendownload file windows 10, kita dapat langsung mendownloadnya di website resmi microsoft disini. Kita dapat mendownload media creation Tool untuk untuk membuat media installnya.

Kita dapat membuat media bootable menggunakan DVD atau flashdisk. Untuk proses setting bootablenya mungkin tidak saya jelaskan lebih lengkap disini.
